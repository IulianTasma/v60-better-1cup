# V60 Better 1 Cup – Interactive Brew Guide

An interactive web page based on **James Hoffmann's Better 1 Cup V60 technique**, built for anyone just getting started with pour-over coffee.

→ **[Open the page](https://iuliantasma.github.io/v60-better-1cup/)**

## What this is

This is a small, mobile-friendly tool that makes it easier to follow James Hoffmann's V60 recipe in real time. It includes:

- A **dose calculator** (15–18g) that scales all water amounts automatically
- A **brew timer** that highlights the current step as you go
- **Audio cues** —> different sounds for bloom, pours, pauses, and the end of the brew
- An **adaptation table** and brewing tips

## Who it's for

Beginners who want a structured starting point for V60 brewing. The goal is to help you get comfortable with the process the timing, the pours, the ratios until you're ready to experiment and develop your own recipe.
